# jtsekret

`jtsekret` — это CLI-утилита для централизованного и безопасного управления личными секретами (пароли, OAuth-токены, API-ключи, токены ботов). Утилита абстрагирует конкретный бэкенд хранилища за единым интерфейсом, реализует локальный шифрованный кэш и позволяет передавать извлечённые секреты в stdin других процессов через Unix-пайп.
